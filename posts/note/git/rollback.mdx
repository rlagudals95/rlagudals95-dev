# 간단한 롤백

## 특정 커밋만 롤백하고 싶을 때
```sh
$ git revert -m 1 <commit hash>
$ git revert -m 1 51045701ca2247dc657020d7774c028c4728adcf
```

## 특정커밋으로 복귀하고 싶을 때
```sh
get reset --hard <commit hash>
git push -f origin head 
```
